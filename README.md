# Bitmovin Analytics Collector

## Prerequisites

* Node >= 8.x
* Yarn package manager: https://yarnpkg.com

## Setup

```
yarn install
```

## Build scripts

```
yarn build:debug
```

```
yarn build:release
```

## Dev mode

```
yarn build:watch // runs webpack in watch mode
```

```
yarn start // starts local http-server
```
