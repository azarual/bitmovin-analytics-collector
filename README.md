# Bitmovin Analytics Collector

## Prerequisites

* Node >= 8.x
* Yarn package manager: https://yarnpkg.com

NOTE: `npm` will also work with the same scripts added the `run ...` command.

## Setup

```
yarn install
```

## Build scripts

```
yarn build:debug
```

```
yarn build:release
```

## Dev mode

```
yarn start // run webpack-dev-server
```
